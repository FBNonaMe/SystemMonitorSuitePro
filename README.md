## 🎯 Возможности

- 🖥 **Мониторинг в реальном времени** — CPU, RAM, диски, сеть, GPU, температура
- 🌐 **Сетевые инструменты** — сканер сети, анализатор трафика, захват пакетов, DNS-анализ
- 🛡 **Безопасность** — управление файрволом, редактор hosts, сканер отладчиков
- 🔧 **Системные утилиты** — реестр, службы, процессы, автозагрузка, планировщик задач
- 🧹 **Оптимизация** — очистка системы, удаление программ, ремонт реестра
- 🛠 **Windows Tweaker** — 24+ настройки Проводника, персонализация, управление обновлениями, UWP-приложениями, активацией и компонентами
- 📊 **Визуализация** — графики загрузки, дашборд, единый интерфейс

## 🧩 Модули

### 📊 Системный монитор
Мониторинг аппаратных ресурсов компьютера в реальном времени:
- Загрузка процессора — общая и по каждому ядру
- Использование оперативной памяти — занято, свободно, процент
- Дисковая активность — скорость чтения и записи
- Сетевая активность — входящий и исходящий трафик
- Обновление данных каждые 2 секунды
- Графики с историей значений

### 📈 Dashboard
Единая панель мониторинга со сводной информацией:
- Виджеты с ключевыми метриками системы
- Общий обзор состояния ПК
- Быстрый доступ к основным показателям

### 🌡 Монитор температуры
Отслеживание температуры компонентов:
- Температура процессора (каждое ядро)
- Температура видеокарты
- Температура накопителей
- Предупреждения при перегреве

### 🎮 GPU Монитор
Детальный мониторинг видеокарты:
- Загрузка GPU в процентах
- Использование видеопамяти
- Температура и частоты
- Графики в реальном времени

### ⚙️ Диспетчер процессов
Расширенное управление запущенными процессами:
- Полный список процессов с PID, именем, потреблением памяти
- Поиск и фильтрация по названию
- Завершение отдельного или нескольких процессов
- Открытие расположения файла процесса
- Подробности о каждом процессе

### 🚀 Менеджер автозагрузки
Управление программами, запускаемыми при старте Windows:
- Программы из реестра (HKCU и HKLM)
- Интеграция с Планировщиком заданий Windows
- Включение и отключение элементов автозагрузки
- Поиск по названию и пути

### ⚙️ Менеджер служб Windows
Управление системными службами:
- Список всех установленных служб с их состоянием
- Запуск, остановка, перезапуск служб
- Изменение типа запуска (автоматический, ручной, отключён)
- Поиск и фильтрация

### 📝 Редактор реестра
Встроенный редактор реестра Windows:
- Навигация по дереву разделов реестра
- Просмотр и редактирование параметров
- Создание и удаление ключей и значений
- Поддержка всех типов данных реестра

### 🔧 Ремонт реестра
Автоматический поиск и исправление ошибок реестра:
- Сканирование на наличие проблем
- Обнаружение несуществующих путей, битых ссылок
- Безопасное исправление найденных ошибок

### 🌐 Сканер сети
Обнаружение устройств в локальной сети:
- Сканирование диапазона IP-адресов
- Определение имени хоста, MAC-адреса
- Отображение списка активных устройств
- Определение производителя по MAC-адресу

### 🌐 Сетевой монитор
Мониторинг сетевой активности в реальном времени:
- Входящий и исходящий трафик
- Активные соединения
- Статистика по протоколам
- Графики загрузки сети

### 🔬 Сетевой анализатор
Глубокий анализ сетевого трафика:
- Захват сетевых пакетов в реальном времени
- DNS-анализ — отслеживание DNS-запросов и ответов
- Анализ активности устройств
- Статистика соединений по IP, портам и протоколам
- Обнаружение подозрительной активности
- Геолокация IP-адресов
- Фильтрация и поиск по пакетам

### 🛡 Менеджер файрвола
Управление брандмауэром Windows:
- Просмотр всех правил файрвола
- Создание новых правил (входящие/исходящие)
- Включение и отключение правил
- Удаление правил
- Фильтрация по направлению, действию и профилю

### 📄 Редактор Hosts
Управление файлом hosts для перенаправления доменов:
- Просмотр и редактирование записей
- Добавление новых записей (IP → домен)
- Блокировка сайтов через перенаправление на 127.0.0.1
- Включение и отключение отдельных записей

### 🧹 Очистка системы
Удаление временных и ненужных файлов:
- Очистка временных файлов Windows
- Очистка кэша браузеров
- Очистка корзины
- Очистка логов
- Отображение размера освобождённого пространства

### 🗑 Удаление программ
Менеджер деинсталляции установленных программ:
- Полный список установленных программ
- Информация: название, издатель, версия, размер
- Запуск деинсталлятора
- Поиск по названию

### 📁 Мониторинг файловой системы
Отслеживание изменений файлов и папок в реальном времени:
- Обнаружение создания, изменения, удаления, переименования файлов
- Автоматическое резервное копирование изменённых файлов
- Настраиваемые маски файлов и каталоги наблюдения
- Журнал всех событий

### 🔌 USB Монитор
Отслеживание подключения и отключения USB-устройств:
- Список текущих USB-устройств
- Уведомления о подключении/отключении
- История событий USB

### 🐛 Сканер отладчиков
Инструмент безопасности для обнаружения средств отладки:
- Обнаружение запущенных отладчиков
- Проверка целостности системы
- Выявление подозрительных процессов

### 🔓 Windows Unlocker
Разблокировка скрытых настроек и функций Windows:
- Доступ к скрытым системным параметрам
- Разблокировка ограниченных функций
- Расширенные настройки системы

### 🔑 Admin Toolkit
Набор административных утилит:
- Быстрый доступ к системным инструментам
- Административные команды
- Системные операции с повышенными правами

### 🛠 Tweaker
Полноценный набор для тонкой настройки Windows. Включает 8 разделов:

**📦 UWP-приложения:**
- Просмотр всех встроенных приложений Windows
- Поиск по названию
- Массовое удаление ненужных системных приложений

**📂 Проводник (24 настройки в 6 категориях):**
- *Отображение файлов* — расширения, скрытые файлы, системные файлы, буквы дисков, флажки выбора, сжатые файлы цветом
- *Навигация* — «Этот компьютер» по умолчанию, полный путь в заголовке и адресе, разворот дерева, отдельные окна
- *Конфиденциальность* — отключение недавних файлов, частых папок, рекомендаций, рекламы в Проводнике
- *Внешний вид* — компактный вид, классическое контекстное меню, скрытие стрелки ярлыков, миниатюры, подсказки
- *Системные* — подтверждение удаления, мастер общего доступа
- *Питание* — отключение гибернации (освобождение дискового пространства)
- Перезапуск Проводника одной кнопкой

**🔄 Центр обновления Windows:**
- Включение/отключение службы обновлений
- Приостановка обновлений на 7 или 30 дней

**🧩 Компоненты Windows:**
- Включение/отключение компонентов (Hyper-V, WSL, .NET Framework и др.)

**🔑 Активация Windows:**
- Проверка статуса активации
- Ввод ключа и активация

**🎨 Персонализация (9 настроек):**
- Тёмная тема приложений и системы, прозрачность
- Панель задач: выравнивание, скрытие поиска, виджетов, Task View, Copilot
- Отключение анимаций

**⚙️ Процессы:**
- Диспетчер процессов с завершением и поиском

**💻 Информация о ПК:**
- ОС, процессор, RAM, диски, GPU, сеть, BIOS

### 🌐 Встроенный браузер
Веб-браузер на базе WebView2 (Chromium):
- Навигация по веб-страницам
- Адресная строка
- Кнопки навигации (назад, вперёд, обновить)

## 🎨 Интерфейс

- **Тёмная тема** — единый стиль оформления во всём приложении
- **Горизонтальная навигация** — все модули доступны через верхнюю панель с прокруткой
- **Адаптивная компоновка** — элементы подстраиваются под размер окна
- **Карточный дизайн** — информация представлена в удобных карточках
- **Переключатели и кнопки действий** — интуитивное управление настройками
- **Индикаторы загрузки** — визуальная обратная связь при длительных операциях
- **Статусная строка** — текущее состояние и результат последней операции
- **Минимизация в трей** — приложение может работать в фоне из области уведомлений

## 🛡 Права администратора

Часть функций требует запуска от имени администратора:

| Функция | Причина |
|---|---|
| Управление файрволом | Изменение правил Windows Firewall |
| Управление службами | Запуск/остановка системных служб |
| Реестр (HKLM) | Системные разделы реестра |
| Управление обновлениями | Остановка системных служб |
| Удаление UWP-приложений | Системные пакеты |
| Отключение гибернации | Команда powercfg |
| Захват пакетов | Доступ к сетевому интерфейсу |
| Редактор hosts | Защищённый системный файл |

Приложение корректно обрабатывает отсутствие прав — функции, не требующие повышения, работают без ограничений.
