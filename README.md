**System Monitor Suite Pro** — десктопное приложение для Windows, объединяющее более 27 инструментов системного администрирования, мониторинга, безопасности и оптимизации в единый интерфейс с тёмной темой. Разработано на .NET 8.0 с использованием WPF и архитектуры MVVM. Доступна в двух вариантах: **Full** (компактная, требует .NET Runtime) и **Portable** (автономная, .NET встроен).

## 🎯 Возможности

- 🖥 **Мониторинг в реальном времени** — CPU, RAM, диски, сеть, GPU, температура
- 🌐 **Сетевые инструменты** — сканер сети, анализатор трафика, захват пакетов, DNS-анализ, обнаружение аномалий, геолокация IP, фильтрация трафика
- 🛡 **Безопасность** — управление файрволом, редактор hosts, сканер отладчиков, логирование системной активности с обнаружением аномалий, подмена Sethc
- 🔧 **Системные утилиты** — реестр, службы, процессы (с принудительным завершением защищённых), автозагрузка (Winlogon, AppInit_DLLs, Shell Extensions), планировщик задач
- 🔬 **Монитор программ** — запуск .exe с отслеживанием порождённых процессов и изменённых служб в реальном времени
- 🧹 **Оптимизация** — очистка системы, удаление программ, ремонт реестра
- 🛠 **Windows Tweaker** — 24+ настройки Проводника, персонализация, управление обновлениями, UWP-приложениями, активацией и компонентами
- 📊 **Визуализация** — графики загрузки, дашборд, единый интерфейс
- 📦 **Два варианта установки** — Full (компактная) и Portable (автономная)

## 🧩 Модули

### 📊 Системный монитор
Мониторинг аппаратных ресурсов компьютера в реальном времени:
- Загрузка процессора — общая и по каждому ядру
- Использование оперативной памяти — занято, свободно, процент
- Дисковая активность — скорость чтения и записи
- Сетевая активность — входящий и исходящий трафик
- Обновление данных каждые 2 секунды
- Графики с историей значений

### 📈 Dashboard
Единая панель мониторинга со сводной информацией:
- Виджеты с ключевыми метриками системы
- Общий обзор состояния ПК
- Быстрый доступ к основным показателям

### 🌡 Монитор температуры
Отслеживание температуры компонентов:
- Температура процессора (каждое ядро)
- Температура видеокарты
- Температура накопителей
- Предупреждения при перегреве

### 🎮 GPU Монитор
Детальный мониторинг видеокарты:
- Загрузка GPU в процентах
- Использование видеопамяти
- Температура и частоты
- Графики в реальном времени

### ⚙️ Диспетчер процессов
Расширенное управление запущенными процессами:
- Полный список процессов с PID, именем, потреблением памяти
- Поиск и фильтрация по названию
- Завершение отдельного или нескольких процессов
- **💀 Принудительное завершение защищённых процессов** — снятие флага Critical Process через `NtSetInformationProcess` с активацией `SeDebugPrivilege`, позволяет завершить даже системные процессы (csrss, lsass, svchost и др.)
- Двойное подтверждение для критических процессов с предупреждением о BSOD
- Определение уровня критичности процесса (Критический / Системный / Обычный)
- **🧊 Заморозка процессов** — приостановка всех потоков процесса через `SuspendThread`
- **Разморозка** — возобновление работы замороженных процессов
- **🔒 Карантин процессов** — изоляция подозрительных процессов
- **🌳 Дерево процессов** — визуализация иерархии родительских и дочерних процессов
- Открытие расположения файла процесса
- Подробности о каждом процессе

### 🚀 Менеджер автозагрузки
Расширенное управление всеми точками автозагрузки Windows:

**Источники автозагрузки:**
- Реестр Run (HKCU) — пользовательские программы
- Реестр Run (HKLM) — машинные программы
- Реестр Run (WOW6432Node) — 32-битные программы на 64-битной системе
- Реестр RunOnce — программы для однократного запуска
- Папка автозагрузки (User) — ярлыки текущего пользователя
- Папка автозагрузки (All Users) — общие ярлыки
- Планировщик задач — задачи с триггерами Logon/Boot

**Winlogon:**
- Shell (оболочка рабочего стола, по умолчанию explorer.exe)
- Userinit (инициализация пользователя при входе)
- TaskMan, AppSetup, Notify, System, UIHost, VMApplet, GpExtensions
- Цветовая индикация уровня риска: Нормальный / Внимание / Опасный / Критический
- Безопасное отключение через бэкап в реестре с восстановлением
- Предупреждение при изменении критических элементов (Shell, Userinit)

**AppInit_DLLs:**
- Мониторинг AppInit_DLLs (x64 и x86) — DLL, загружаемые в каждый процесс
- Включение/отключение через LoadAppInit_DLLs

**Shell Extensions:**
- ShellExecuteHooks — хуки выполнения команд оболочки
- Browser Helper Objects (BHO) — расширения браузера IE/Edge Legacy

**Управление:**
- Включение/отключение через стандартный механизм Windows (`StartupApproved`) — совместимо с Диспетчером задач
- Фильтрация по категориям и полнотекстовый поиск
- Удаление элементов автозагрузки
- Открытие расположения файла

### 📅 Планировщик задач
Полноценное управление Планировщиком заданий Windows:
- Просмотр всех запланированных задач с подробной информацией
- Поиск и фильтрация по имени, пути и описанию
- Включение и отключение задач
- Ручной запуск задач по требованию
- Удаление задач с подтверждением
- Отображение количества найденных задач и статуса операций

### ⚙️ Менеджер служб Windows
Управление системными службами:
- Список всех установленных служб с их состоянием
- Запуск, остановка, перезапуск служб
- Изменение типа запуска (автоматический, ручной, отключён)
- Поиск и фильтрация

### 📝 Редактор реестра
Встроенный редактор реестра Windows:
- Навигация по дереву разделов реестра
- Просмотр и редактирование параметров
- Создание и удаление ключей и значений
- Поддержка всех типов данных реестра

### 🔧 Ремонт реестра
Автоматический поиск и исправление ошибок реестра:
- Сканирование на наличие проблем
- Обнаружение несуществующих путей, битых ссылок
- **Интеллектуальный анализ уверенности** — каждая проблема получает оценку (High / Medium / Low) на основе защищённости ключа, категории ошибки, наличия связанных процессов
- Автоматический выбор стратегии исправления: удаление, удаление с бэкапом, отключение или только предпросмотр
- Защита критических системных ключей от случайного изменения
- **Полное резервное копирование** — создание бэкапов перед каждым исправлением
- Откат отдельных изменений или целых сессий ремонта
- История ремонтов с группировкой по сессиям
- Очистка старых резервных копий по сроку хранения

### 🌐 Сканер сети
Обнаружение устройств в локальной сети:
- Сканирование диапазона IP-адресов
- Определение имени хоста, MAC-адреса
- Отображение списка активных устройств
- Определение производителя по MAC-адресу (OUI-база)
- Уведомления о новых устройствах в сети
- Корреляция MAC ↔ IP адресов

### 🌐 Сетевой монитор
Мониторинг сетевой активности в реальном времени:
- Входящий и исходящий трафик
- Активные соединения
- Статистика по протоколам
- Графики загрузки сети

### 🔬 Сетевой анализатор
Глубокий анализ сетевого трафика с профессиональными средствами обнаружения угроз:

**Захват и анализ пакетов:**
- Захват сетевых пакетов в реальном времени
- Статистика по протоколам (TCP, UDP, ICMP), портам и IP-адресам
- Фильтрация и поиск по пакетам
- Подсчёт пакетов в секунду и скорости передачи данных

**DNS-анализ:**
- Отслеживание DNS-запросов и ответов с историей
- Агрегация статистики по доменам (количество запросов, уникальные устройства, разрешённые IP)
- DNS-отслеживание по устройствам — какие домены запрашивает каждое устройство
- Обратное разрешение IP → домен из DNS-ответов
- **Обнаружение DNS-туннелирования** — анализ аномально длинных доменов с высокой энтропией Шеннона
- **Обнаружение DGA** — выявление алгоритмически сгенерированных доменов (C2-связь)
- Отслеживание NXDOMAIN-ответов

**Обнаружение аномалий:**
- **Обнаружение всплесков трафика** — оповещение при >500 пакетов за 2 секунды
- **Обнаружение сканирования портов** — детекция SYN-сканирования по >15 уникальным портам
- **Обнаружение избыточных соединений** — оповещение при >50 соединений с одного IP
- **Обнаружение Beaconing** — выявление периодических соединений (C2-паттерн) через статистический анализ джиттера

**Инспекция трафика устройств:**
- Профилирование трафика по каждому устройству (по MAC или IP)
- Все соединения устройства с направлением, объёмом данных и длительностью
- Агрегация направлений трафика с геолокацией (страна, город, провайдер)
- Журнал последних коммуникаций с цветовой кодировкой протоколов

**Геолокация IP:**
- Определение страны, города, региона, провайдера, организации, AS-номера
- Отображение флагов стран
- Автоматическое определение локальных/приватных адресов
- Кэширование результатов для производительности

**Система фильтрации:**
- Белый и чёрный списки IP-адресов и доменов
- Суффиксное сопоставление доменов (блокировка `example.com` блокирует и `tracker.example.com`)
- Импорт и экспорт списков из текстовых файлов
- Взаимное исключение: добавление в белый список автоматически убирает из чёрного

**📂 Проводник (24 настройки в 6 категориях):**

- *Системные* — подтверждение удаления, мастер общего доступа
- *Питание* — отключение гибернации (освобождение дискового пространства)

### 📋 Журнал системной активности
Мониторинг изменений в системе в реальном времени с автоматическим обнаружением подозрительной активности:

**Мониторинг процессов:**
- Отслеживание запуска и завершения процессов через WMI (`Win32_ProcessStartTrace` / `Win32_ProcessStopTrace`)
- Обнаружение запуска известных хакерских инструментов (mimikatz, lazagne, psexec, procdump и др.)
- Анализ путей запуска — запуск из `\Temp\`, `\Downloads\`, `\AppData\` помечается как подозрительный
- Обнаружение двойных расширений (например, `file.pdf.exe`)
- Анализ родительских процессов — запуск неизвестных процессов из cmd/powershell
- Игнорирование известных системных процессов (svchost, explorer, csrss и др.)

**Мониторинг служб:**
- Отслеживание изменений состояния служб через WMI
- Оповещение при остановке служб безопасности (Windows Defender, Firewall, Security Center)
- Отслеживание остановки служб с автозапуском

**Мониторинг автозагрузки:**
- Периодическая проверка ключей реестра `HKCU\...\Run` и `HKLM\...\Run` (каждые 10 секунд)
- Уведомление о новых записях автозагрузки с оценкой подозрительности

**Визуализация и фильтрация:**
- Цветовая кодировка событий: 🔴 красный — подозрительно, 🟡 жёлтый — предупреждение, 🟢 зелёный — информация
- Фильтрация по категории (Процессы / Службы / Реестр / Файловая система / Сеть / Система)
- Фильтрация по уровню серьёзности (Информация / Предупреждение / Подозрительно)
- Полнотекстовый поиск по описанию событий
- Счётчики событий: общее количество, подозрительные, предупреждения
- Лимит 5000 записей в памяти, 2000 в интерфейсе

### 🔬 Монитор программ
Запуск выбранной программы с отслеживанием всех изменений в системе в реальном времени:
- Выбор .exe через проводник и запуск с мониторингом
- Автоматический снимок процессов и служб перед запуском (baseline)
- Обнаружение новых запущенных процессов с PID
- Обнаружение остановленных процессов
- Отслеживание появления, удаления и изменения статуса служб
- Автоматическое обнаружение завершения мониторируемой программы
- Таблица событий с колонками: Время (мс), Тип, Действие, Имя, Детали
- Индикатор LIVE при активном мониторинге
- Кнопки: Запустить мониторинг / Остановить / Очистить журнал
- Настраиваемый интервал опроса (1.5 сек)

### 🔑 Sethc Replacer
Подмена программы специальных возможностей (Sticky Keys) на экране блокировки:
- Замена sethc.exe на cmd.exe, PowerShell, Диспетчер задач, Проводник, Блокнот, Regedit, Панель управления
- Выбор произвольного .exe через проводник
- Автоматическое создание резервной копии (sethc.exe.bak)
- Восстановление оригинального sethc.exe из бэкапа
- Метод: takeown → icacls → copy (требует прав администратора)
- Вызов через 5× нажатие Shift на экране блокировки

### 🌐 Встроенный браузер
Веб-браузер на базе WebView2 (Chromium) со встроенным анализатором:
- Навигация по веб-страницам
- Адресная строка
- Кнопки навигации (назад, вперёд, обновить)
- **SEO-анализ сайтов** — проверка title, meta description, H1-заголовков, viewport, canonical URL
- **Аудит безопасности** — проверка заголовков HSTS, X-Frame-Options, CSP, X-Content-Type-Options, X-XSS-Protection
- **Анализ производительности** — время загрузки, размер страницы, количество ресурсов (CSS, JS, изображения)
- Оценки и рекомендации по каждому параметру

## 📥 Установка и запуск

### Что нужно установить перед запуском

1. **[.NET 8.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)** — требуется только для **Full-версии** (Portable-версия уже включает .NET)
2. **[WebView2 Runtime](https://developer.microsoft.com/microsoft-edge/webview2/)** — нужен для встроенного браузера (обычно уже установлен с Windows 11 или Edge)
3. **[Npcap](https://npcap.com/#download)** — требуется для модуля захвата сетевых пакетов (сетевой анализатор). При установке отметьте галочку «Install in WinPcap API-compatible mode»

### Запуск

1. Скачайте установщик `SystemMonitorSuite_Setup_v1.1.0.exe` из раздела [Releases](../../releases)
2. Запустите установщик и выберите тип установки:
   - **Полная установка** — компактная (~26 МБ), требует установленного .NET 8.0 Desktop Runtime
   - **Portable** — автономная (~80 МБ), .NET Runtime встроен, работает без дополнительных зависимостей
3. После установки запустите приложение **от имени администратора** для полного функционала

> ⚠️ Если при запуске Full-версии появляется ошибка — убедитесь, что установлен .NET 8.0 Desktop Runtime.  
> Portable-версия работает без установленного .NET.

## 🛡 Права администратора

Часть функций требует запуска от имени администратора:

| Функция | Причина |
|---|---|
| Управление файрволом | Изменение правил Windows Firewall |
| Управление службами | Запуск/остановка системных служб |
| Реестр (HKLM) | Системные разделы реестра |
| Управление обновлениями | Остановка системных служб |
| Удаление UWP-приложений | Системные пакеты |
| Отключение гибернации | Команда powercfg |
| Планировщик задач | Управление системными задачами |
| Захват пакетов | Доступ к сетевому интерфейсу |
| Редактор hosts | Защищённый системный файл |
| Принудительное завершение процессов | SeDebugPrivilege, NtSetInformationProcess |
| Журнал системной активности | WMI-подписки на системные события |
| Sethc Replacer | takeown/icacls на системный файл |
| Менеджер автозагрузки (HKLM, Winlogon) | Системные разделы реестра, StartupApproved |

Приложение корректно обрабатывает отсутствие прав — функции, не требующие повышения, работают без ограничений.
